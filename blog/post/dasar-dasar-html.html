DASAR DASAR HTML

Diposting: 17 April 2025 oleh Marpel

programming

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML berfungsi untuk menyusun struktur dan tampilan konten di browser. Artikel ini akan membahas dasar-dasar HTML yang wajib dipahami oleh pemula.

1. Teks dan Format Dasar HTML

  • <h1> sampai <h6>: Menampilkan judul dengan ukuran berbeda
  • <p>: Membuat paragraf
  • <br>: Garis baru (line break)
  • <b>, <i>, <u>: Menebalkan, memiringkan, dan menggarisbawahi teks

2. Menyisipkan Gambar, Audio, dan Video

<img> digunakan untuk gambar, <audio> untuk suara, dan <video> untuk video.

3. Tabel

Digunakan untuk menyusun data dalam baris dan kolom.

  • <table>: Elemen utama tabel
  • <tr> (table row): Baris dalam tabel
  • <td> (table data): Sel data dalam baris
  • <th> (table header): Judul kolom

4. Tombol

Tombol bisa dibuat dengan <button> atau <input type="button">, dan digunakan untuk aksi seperti mengirim form atau interaksi JavaScript.

5. Tautan

<a> adalah tag untuk membuat tautan ke halaman lain, file, atau bagian halaman yang sama.

6. Tambahan: Formulir dan Input

Digunakan untuk menerima input dari pengguna, seperti <input type="text">, <email>, dan sebagainya.

7. Struktur Dasar HTML

HTML terdiri dari struktur utama:
<!DOCTYPE html>, <html>, <head>, dan <body>.
